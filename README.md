## Installation

Clone repository and change directory 

```
git clone https://github.com/zyz-airdrops/w3bflix-claimer.git
```

```
cd w3bflix-claimer
```
Install the required libraries:

```
 python -m pip install -r requirements.txt
```
Add your query id to the file:
```
nano data.txt
```
